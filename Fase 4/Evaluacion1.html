SINTAXIS DE MEDIA QUERIES

la media queries consisten de un tipo de medio opcional y una o mas expresiones de caracteristicas de medio varias consultas se pueden combinar utilizando operadores lógicos. No distinguen entre mayusculas.

El resultado de la consulta es verdadero cuando eltipo de medio si se especifica coincide con el dispositivo en el que se esta mostrando el documento y todas las expreciones en el media query son "verdaderos" en este caso se aplica los estilos correspondientes, siguiendo las reglas usuales de cascada.
