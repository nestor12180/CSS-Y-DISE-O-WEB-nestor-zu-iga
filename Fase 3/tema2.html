Tema 2: ¿Por que es fundamental el HTML?

El html es parte fundamental de las normas web, en conjunto con otras tecnologias como CSS o el JAVA SCRIP,

ya que este lenguaje es la base para construir paginas en internet ya que es importante conocerlo para poder crear nuesta proopia pagina WEB

¿Que es y para que sirve el HTML

HTML son las siglas en ingles para HYPERTEXT MARKUP LANGUAGE, que se traduce como lenguaje de marcacion de hipertexto

A pesar de tener un monbre complicado, el HTML no es mas que un lenguaje usado para crear paginas webs por medio de marcadores (TAG) y atributos,

que definen como el contenido va ser presentado en un navegador web
